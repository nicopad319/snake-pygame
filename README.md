# Snake — Pygame

A terminal Snake game I originally built in C++, ported to Python using pygame.
Includes three difficulty modes.

## Gameplay

- **Easy** — wall collisions only
- **Normal** — wall + self collisions  
- **Hard** — normal + random barriers spawn after every fruit collected

## Controls

| Key | Action |
|-----|--------|
| W | Up |
| A | Left |
| S | Down |
| D | Right |
| ESC | Quit |

## Run

```
pip install pygame
py -3.11 main.py
```
